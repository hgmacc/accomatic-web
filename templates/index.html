Compare your models.

Accomatic can be run on your local machine - download using pip.

As simple as:

pip install accomatic
model1 = pickle(csvData1)
model2 = pickle(csvData2)
observations = pickle(csvObsData)
accomatic(observations, model1, model2)

What does Accomatic do?

This software performes a suite of comparisons, including:

Temporal variability - a seasonal comparison
Spatial variability - shifts in terrain type
Accordance measure selection - RMSE vs E_1
Handles periods of missing observational data
